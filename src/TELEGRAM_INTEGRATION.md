# Интеграция с Telegram Web App

## Шаги для запуска в Telegram

### 1. Создание бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### 2. Настройка Web App

1. Отправьте команду `/mybots` в BotFather
2. Выберите вашего бота
3. Нажмите "Bot Settings" → "Menu Button"
4. Выберите "Edit menu button URL"
5. Введите URL вашего развёрнутого приложения

### 3. Деплой приложения

Приложение необходимо развернуть на HTTPS хостинге. Варианты:

- **Vercel** (рекомендуется)
- **Netlify**
- **GitHub Pages**
- **Cloudflare Pages**

### 4. Тестирование

1. Откройте вашего бота в Telegram
2. Нажмите на кнопку Menu (иконка ≡)
3. Приложение откроется в Telegram Web App

## Возможности Telegram SDK

### Доступные функции

- `hapticFeedback` - вибрация при действиях
- `hapticNotification` - уведомления (success/error/warning)
- `shareLink` - поделиться ссылкой в Telegram
- `user` - данные пользователя Telegram
- `isInTelegram` - проверка запуска в Telegram

### Особенности

- Приложение автоматически адаптируется под тему Telegram
- Цвета настроены под дизайн приложения (#F5E0B7)
- Поддержка тактильной обратной связи
- Интеграция с профилем пользователя

## Локальное тестирование

Для тестирования локально с Telegram Web App:

1. Используйте ngrok или подобный сервис для HTTPS туннеля
2. Настройте Menu Button URL на ngrok адрес
3. Откройте бота в Telegram

```bash
# Пример с ngrok
ngrok http 3000
# Используйте HTTPS URL в настройках бота
```

## Безопасность

- Проверяйте `initData` на бэкенде для валидации пользователей
- Не храните чувствительные данные в localStorage
- Используйте HTTPS для всех запросов
